FEATURES

What this app
includes

User Auth

Register and login with bcrypt-hashed passwords. Passwords are never stored as plain text in the database.

Contact Form

Contact form that saves name, email and message directly into MySQL database using PHP backend.

User Dashboard

After login the user sees a personal dashboard with their name and basic stats. Has a logout button too.

Admin Panel

Admin can view all contact form submissions in a table. Can search by name or email and view full messages.
